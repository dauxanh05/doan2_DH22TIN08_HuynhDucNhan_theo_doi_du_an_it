## Frontend

| Tech | Ly do |
|------|-------|
| React 18 + TypeScript | Framework quen thuoc, ecosystem lon |
| Vite | Nhanh hon CRA, HMR tot |
| TailwindCSS + Headless UI | Utility-first, nhanh, customizable |
| Zustand | Nhe hon Redux, API don gian, persist middleware |
| TanStack Query | Caching, refetching, loading states tu dong |
| React Hook Form + Zod | Performance tot (uncontrolled), schema validation |
| React Router v6 | Standard routing, nested routes |
| @dnd-kit | Accessible drag-drop, nhe hon react-beautiful-dnd |
| Recharts | Don gian, du cho pie + bar charts |
| date-fns | Nhe hon moment.js, tree-shakable |
| Lucide React | Consistent icons, nhe |
| Socket.io-client | Pair voi server Socket.io |
