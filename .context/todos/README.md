# Todos

> Pending ideas va quick tasks chua lam.
> Moi todo la 1 file rieng trong `pending/`.
> Move sang `done/` khi hoan thanh.

## Cach dung

### Tao todo moi

Tao file `pending/NNN-ten-ngan.md` voi format:

```markdown
# Ten y tuong

- **Priority:** LOW / MEDIUM / HIGH
- **Category:** auth / workspace / frontend / infra / ...
- **Created:** YYYY-MM-DD
- **Related:** requirement ID hoac branch (neu co)

## Mo ta

Chi tiet y tuong, tai sao can lam, goi y cach lam.
```

### Hoan thanh todo

Move file tu `pending/` sang `done/`:
```bash
mv pending/001-ten.md done/001-ten.md
```

## Numbering

- Bat dau tu `001`
- Tang dan: `001`, `002`, `003`...
